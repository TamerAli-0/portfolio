# Tamer Altaweel — Developer Portfolio

A modern, animated portfolio website built with vanilla HTML, CSS, and JavaScript. Features a dark theme with interactive particle effects, smooth scroll animations, and a fully responsive design.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

## Features

- **Interactive Particle Background** — Canvas-based particles that react to mouse movement and connect with proximity lines
- **Cursor Glow** — Radial gradient that follows the cursor across the page
- **Typewriter Effect** — Rotating text animation showcasing different skills
- **Scroll Reveal Animations** — Elements fade and slide into view using Intersection Observer
- **Animated Skill Bars** — Progress bars that fill on scroll with gradient styling
- **Counter Animation** — Stats count up from zero when scrolled into view
- **Glassmorphism UI** — Semi-transparent cards with backdrop blur and subtle borders
- **Code Window** — Syntax-highlighted code block in the hero section
- **Floating Label Form** — Contact form with animated labels and underline transitions
- **Fully Responsive** — Adapts to mobile, tablet, and desktop with a collapsible nav menu

## Sections

| Section | Description |
|---------|-------------|
| **Hero** | Introduction with typewriter, stats, and code window |
| **About** | Background, experience badge, and contact details |
| **Skills** | Frontend, Backend, and Tools with animated progress bars |
| **Projects** | Featured work with hover overlays and tech tags |
| **Contact** | Contact form and direct links (email, GitHub) |

## Getting Started

Clone the repo and open `index.html` in a browser — no build tools required.

```bash
git clone https://github.com/TamerAli-0/portfolio.git
cd portfolio
# Open index.html in your browser
```

## Live Demo

Enable GitHub Pages in **Settings > Pages > Source: master** to host at:

```
https://tamerali-0.github.io/portfolio/
```
